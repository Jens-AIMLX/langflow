## üö® **TROUBLESHOOTING GUIDE**

### **Common Issue 1: Rust Compilation Appears Stuck**

**Symptom**: Build appears stuck at "712/715" packages
**Solution**:
- ‚úÖ **THIS IS NORMAL** - final linking phase takes 10-30 minutes
- ‚úÖ Do NOT interrupt the process
- ‚úÖ Monitor file size growth of `nautilus_pyo3.dll` in `nautilus_core/target/release/deps/`

### **Common Issue 2: Memory Issues During Build**

**Symptom**: Build fails with out-of-memory errors or "killed" messages
**Solutions**:
- ‚úÖ Close other applications, ensure 16GB+ available RAM
- ‚úÖ Reduce parallel compilation: `export CARGO_BUILD_JOBS=1`
- ‚úÖ Use swap file if necessary
- ‚úÖ Restart computer and try again

### **Common Issue 3: Import Errors After Build**

**Symptom**: `ModuleNotFoundError` after build
**Solutions**:
- ‚úÖ Ensure Python build (Step 4) completed successfully
- ‚úÖ Check that `nautilus_trader/core/nautilus_pyo3.cp312-win_amd64.pyd` exists
- ‚úÖ Verify virtual environment is activated
- ‚úÖ Run `python build.py` again if files are missing

### **Common Issue 4: Cython Version Issues**

**Symptom**: Build fails with Cython compilation errors
**Solutions**:
- ‚úÖ Ensure EXACT Cython version: `pip install "Cython==3.1.0a1"`
- ‚úÖ Verification: `python -c "import Cython; print(Cython.__version__)"`
- ‚úÖ Uninstall and reinstall if wrong version

### **Common Issue 5: MSVC Compiler Not Found (Windows)**

**Symptom**: "Microsoft Visual C++ 14.0 is required"
**Solutions**:
- ‚úÖ Install Visual Studio Build Tools 2019 or later
- ‚úÖ Download: https://visualstudio.microsoft.com/downloads/#build-tools-for-visual-studio-2019
- ‚úÖ Verification: `where cl.exe` should show compiler path
- ‚úÖ Restart computer after installation

### **Common Issue 6: Poetry Installation Issues**

**Symptom**: Poetry commands fail or dependencies not found
**Solutions**:
- ‚úÖ Reinstall Poetry: `pip uninstall poetry && pip install poetry`
- ‚úÖ Clear Poetry cache: `poetry cache clear --all pypi`
- ‚úÖ Use pip fallback: `pip install -e .`

### **Common Issue 7: Terminal Hangs with ^C**

**Symptom**: PowerShell terminals hang and don't execute commands
**Solutions**:
- ‚úÖ Close all PowerShell windows
- ‚úÖ Open new PowerShell as Administrator
- ‚úÖ Restart computer if problem persists
- ‚úÖ Use Command Prompt instead of PowerShell
